# SNST_Project
Проект для конкурса "Инженеры будущего", 2026 год.

Программа представляет собой простой генератор адиосигнала.
На данный момент доступна первая и единственная готовая версия программы, собранная под компьютеры Mac с чипом Apple Silicon.
